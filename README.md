# Doom Scroll

A visual metaphor for doom scrolling — a red swing boat swings in the dark as you scroll. The faster you scroll, the wilder the swing. Stop, and it slowly comes to rest.

## What it does

- A swing boat sits centered on screen, suspended from ropes
- Scrolling the page accelerates the swing
- Speed decays naturally when you stop scrolling
- Messages appear below the boat reflecting how far gone you are
- The boat glows red at high speed

## Run it

Just open `index.html` in any browser — no build step, no dependencies.

## Why

Doom scrolling is a habit that erodes attention and time without giving anything back. This small piece tries to make that feeling visible.
